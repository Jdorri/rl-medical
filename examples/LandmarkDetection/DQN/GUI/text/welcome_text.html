Welcome!


Welcome to the Anatomical Landmark Detection and Demonstration Interface! ALADDIN is an interface for visualising landmark detection reinforcement learning (RL) agents trained on brain MRI, cardiac MRI, and fetal ultrasound 3D images. ALADDIN also facilitates Human-in-the-Loop (HITL) experimentation by facilitating human interaction with the enviroment and storing the results.


Automatic mode


In Automatic mode, you can visualise the learned policy of the agent on 2D projections and within a 3D volume.

Browse mode


In Browse mode, the user can select one of six possible actions to freely navigate through the 3D enviroment.

HITL mode


HITL mode allows a human expert to record steps in the enviroment towards an anatomical landmark. This data is automatically stored for further use in HITL experiments.

For more information about installation, problem specification, and paper references, please visit https://github.com/ollenilsson19/rl-medical
