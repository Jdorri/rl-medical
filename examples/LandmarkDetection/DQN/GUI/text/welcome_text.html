Welcome!


Welcome to Anatomical Landmark Detection and Demonstration Interface! ALADDIN is a interface for visualising landmark detection reinforcement learning (RL) agents trained on brain MRI, cardiac MRI, and fetal ultrasound 3D images. ALADDIN also facilitates Human-in-the-Loop (HITL) experiments by storing human steps in the enviroment.


Automatic Mode


In Automatic mode, you can simultaneously visualise the learned policy of the agent on 2D projections and within a 3D volume.

Browse Mode


In Browse mode, the user can use one of the six possible actions to freely navigate through the 3D enviroment.

HITL Mode


HITL mode allows an human expert to record steps in the enviroment towards an anatomical landmark. This data is automatically stored for further use in HITL experiments.

For more information about installation, problem specification, and paper references, please visit https://github.com/ollenilsson19/rl-medical
