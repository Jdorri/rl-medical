Browse Mode

(For more information: Ctrl+H)



1. Load data: choose default or browse custom data.
2. Enable HITL: toggle checkbox to notify to store user interaction data.
3. Navigation: press Human Actions arrow to navigate agent until Error = 0.
4. Next Image: press next image to annotate next sequence of image.
5. Save data: toggle off Enable HITL checkbox or close GUI to save user interaction.
